## Configuration

**Prerequisites:** 

- [Docker](https://www.docker.com/products/docker-desktop/)
- [Docker Compose](https://docs.docker.com/compose/)

 Install Docker and Docker Compose on your system. Docker Desktop works on macOS and Windows and Linux users can install Docker Engine and Docker Compose via their package manager.

### Environment Variables:

**Stopping the Application:** Run `docker-compose down` to stop all containers.
